# bbf-institute

واجهة موقع **معهد بناء المستقبل المشرق** مبنية بـ **React + Vite + TailwindCSS**.

## تشغيل محلياً

```bash
npm install
npm run dev
```

## بناء المشروع

```bash
npm run build
npm run preview
```

## النشر على Netlify

- **Build command:** `npm run build`
- **Publish directory:** `dist`

> ملاحظة: ملف `public/_redirects` مضاف لدعم React Router ومنع خطأ 404 عند تحديث الصفحة.
